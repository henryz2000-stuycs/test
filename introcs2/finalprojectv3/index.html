What is it?

Handy Hotspot NYC is a website created by Gilvir Gill and Henry Zheng. Written in Python, it takes the location given by the user and compares distances between it and a database of hotspots throughout NYC. Once it finds the closest one by using Google Maps Geocode API to get coordinates and then using math to fnd the distance between 2 given coordinate points, it will then give the user directions to the closest location.

Usage:

Enter an address into the text field.
The computer will take your address, and it will find the closest Wi-Fi hotspot.
